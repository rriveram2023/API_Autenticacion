# API_Autenticacion

Repositorio separado para la API de autenticacion y el frente nginx/oauth2-proxy.

Incluye:
- servicio FastAPI en services/auth_api
- introspeccion de sesion, identidad y grupos
- configuracion nginx para publicar https://e3display.com
- scripts de soporte para cookie secret, hosts y validacion HTTPS

Punto de entrada:
- main.py
